# About Tax Structure Watch

## Mission

Tax Structure Watch maps the distributional impact of every state's tax structure using primary-source data with genuinely nonpartisan framing. We present competing analytical frameworks — Tax Foundation (center-right) and ITEP (center-left) — side by side, with transparent discussion of their differing assumptions, linked back to primary federal and state data sources.

## Editorial Framework

**Voice**: Authoritative but accessible. Investigative journalism, not academic paper. The reader is a smart non-specialist — organizer, local official, journalist, concerned citizen.

**Core methodology**: "Follow the money" — trace financial flows through public systems using primary-sourced data to surface structural contradictions.

**Nonpartisan commitment**: We present competing frameworks transparently. We never editorialize on which framework is "correct." We show the data, show the assumptions, show the limitations, and let users evaluate.

**The test**: Can a Republican delegate in Iowa and a Democratic delegate in Minnesota both use this platform productively? If yes, the framing is right.

## Data Standards

- All data from public, primary sources (Census Bureau, IRS, BEA, state revenue departments)
- Automated, reproducible data pipelines
- Transparent methodology with cited sources
- Every calculated metric has a methodology page explaining assumptions and limitations
