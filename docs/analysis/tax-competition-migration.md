# Tax Competition and Migration: What the Data Shows

!!! info "Research in Progress"
    This analysis is currently under development. The data pipeline is operational
    and source collection is underway. Expected publication: Q3 2026.

## What This Analysis Will Cover

IRS Statistics of Income migration data tracks every address change filed on a tax return since 1990. This data is the empirical backbone of the tax competition debate, showing how many people (and how much adjusted gross income) moves between states each year. The critical question is causation: do people move because of taxes, or do other factors (housing costs, jobs, climate, remote work) drive the flows?

This analysis will build a comprehensive picture of interstate migration patterns using the full IRS SOI county-to-county dataset. We will pair migration flows with state tax policy changes, controlling for economic conditions, housing costs, and labor market dynamics. The goal is not to resolve the causation debate definitively, but to present the data clearly enough that readers can evaluate the competing claims for themselves.

We will pay particular attention to the post-2020 period, when remote work expanded the set of people who could relocate without changing employers — potentially increasing the elasticity of migration to tax differentials.

## Data Sources

- IRS Statistics of Income Migration Data (county-to-county flows, 1990–present)
- Census Bureau annual population estimates
- Bureau of Economic Analysis personal income data by state

## Preliminary Findings

The IRS SOI migration data pipeline is currently under development. Once operational, it will provide county-level inflow and outflow data covering returns filed, exemptions claimed, and adjusted gross income transferred between jurisdictions.
